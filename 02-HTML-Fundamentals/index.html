The Basic languaje of the web: HTML

Posted by Laura Jones on Monday, June 21st 2027

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and Javascript. These are the lenguages of the web

In this post, let´s focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for HyperText Markup Language. It´s a markup langauge that web developers use to structure and describe the content of a webpage (not a programming language)

HTML consist of elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites

in HTML, each elements is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
